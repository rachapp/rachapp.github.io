Advanced Free-Electron Laser (FEL) Simulations

Lead Researcher

Project Goal

To leverage high-performance computing to simulate and explore novel operational modes and new physical regimes of Free-Electron Lasers. The primary objective is to push the boundaries of coherent light source performance and uncover new operational possibilities.

My Contributions

  • Developing and running complex FEL simulation codes to model laser dynamics.
  • Analyzing large-scale simulation data to identify and characterize new modes of operation.
  • Utilizing the ThaiSC High-Performance Computing (HPC) facility to execute computationally intensive tasks and manage massive datasets.

Technologies

Fortran/C++, Python, High-Performance Computing (HPC), Parallel Programming (MPI/OpenMP), Scientific Data Analysis Libraries.

Python-Based Beamline Control & Data Acquisition

Lead Developer

Project Goal

To develop a modern, robust, and user-friendly software suite for controlling beamline components and managing the entire data lifecycle at SLRI, from acquisition and motion control to real-time processing.

My Contributions

  • Designing and implementing a comprehensive Python application for integrated motion control and data acquisition systems.
  • Creating an intuitive Graphical User Interface (GUI) using the PyQt6 framework to simplify user interaction with complex hardware.
  • Developing scripts for on-the-fly data processing and visualization to provide immediate experimental feedback.

Technologies

Python, PyQt6, EPICS, Motion Controllers, Data Acquisition Hardware, NumPy, Pandas.

Double-Crystal Monochromator (DCM) Development & Commissioning

Commissioning Scientist

Project Goal

To lead the complete development, integration, and commissioning of a critical beamline component—the Double-Crystal Monochromator (DCM)—ensuring its precise and reliable operation for user experiments.

My Contributions

  • Developed the complete Python-based GUI for DCM control, calibration, and diagnostics.
  • Implemented and rigorously verified the precision of the crystal movements and rotations to meet stringent operational requirements.
  • Integrated a laser-based beam position monitoring system for active feedback and enhanced stability.
  • Conducted full commissioning tests to validate optical and mechanical performance against design specifications.

Technologies

Python, PyQt6, Precision Motion Control, Optical Alignment, Laser Systems, Vacuum Technology, Interferometry.
